Tweji 2wesme

Spice up your website with Twitter emojis without going crazy finding their Unicode equivalent.

What's Twemoji 2wesome?

Twemoji 2wesome is a maintained fork of Elle Kasai's Twemoji Awesome made by Francesco Rosi with more features. Twemoji is an official library made by Twitter to use Twitter emojis in any website.

Why Twemoji 2wesome?

Forked from Twemoji Awesome AND Font Awesome

Since Twemoji 2wesome is also forked from Font Awesome, you get all the Font Awesome features.
Want your emojis to spin?
Or want them rotated by 90 degrees?

Newer emoji list

Twemoji 2wesome uses a newer version of Twemoji Possum made by sptndc, which is the list of Twemoji emojis' name. There are 3 new emoji definitions:

  • "fu" for "middle-finger"
  • "simple-smile" for "slightly-smiling-face"
  • "metal" for "sign-of-the-horns"

Insanely FAST

The emojis are served from Statically, a Multi-CDN powered by Fastly, Cloudflare, bunny.net, Amazon CloudFront and Google Cloud. In other words: it's very very fast.

Install

Right now, Twemoji 2wesome can only be served via GitHub. I recommend serving it from Statically:

<link rel="stylesheet" href="https://cdn.statically.io/gh/francescorosi27/twemoji-2wesome/master/dist/twemoji-awesome.css" />

However I hope I can host this on cdnjs in the future.
